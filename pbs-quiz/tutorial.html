TextThresher Tutorial

Hi! This is a demo project that shows how TextThresher will work using the PyBossa framework on Crowdcrafting.org.

You will be shown a series of phrases and asked to answer questions about those phrases.

The phrase will be partly highlighted. If the highlight should be larger in order to properly reflect your answer, you should adjust the highlight using the triangular handles on the ends of the highlight.
